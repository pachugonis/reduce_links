# Инструкция по установке URL Shortener на Ubuntu 24

## 📋 Требования

- **Операционная система:** Ubuntu 24.04 LTS (чистая установка)
- **Права:** root-доступ (sudo)
- **Домен:** желательно иметь привязанный домен (для SSL)

## 📦 Что входит в архив

```
reduce_links.zip
├── install.sh          ← Скрипт автоматической установки
├── database.sql        ← Структура базы данных
├── includes/           ← PHP классы
├── public/             ← Веб-файлы
└── assets/             ← CSS и статика
```

## 🚀 Быстрая установка

### Шаг 1: Подготовка сервера

Подключитесь к серверу по SSH:

```bash
ssh root@your-server-ip
```

### Шаг 2: Загрузка архива

Скопируйте архив на сервер любым удобным способом:

**Вариант A — через SCP (с локального компьютера):**
```bash
scp reduce_links.zip root@your-server-ip:/root/
```

**Вариант B — через wget (если архив доступен по ссылке):**
```bash
cd /root
wget https://your-domain.com/reduce_links.zip
```

**Вариант C — через GitHub:**
```bash
cd /root
wget https://github.com/pachugonis/reduce_links/releases/download/v1.0/reduce_links.zip
```

### Шаг 3: Распаковка архива

```bash
cd /root
unzip reduce_links.zip -d reduce_links
cd reduce_links
```

### Шаг 4: Запуск установки

```bash
chmod +x install.sh
sudo ./install.sh
```

## ⚙️ Что делает скрипт установки

Скрипт автоматически выполнит:

1. ✅ Обновление системных пакетов
2. ✅ Установка Apache, PHP 8.x, MySQL 8.x
3. ✅ Установка всех необходимых PHP-расширений
4. ✅ Генерацию случайных безопасных паролей
5. ✅ Создание базы данных и пользователя MySQL
6. ✅ Настройку конфигурации PHP
7. ✅ Настройку Apache виртуального хоста
8. ✅ Включение mod_rewrite
9. ✅ Получение SSL-сертификата Let's Encrypt (если домен доступен)
10. ✅ Настройку фаервола UFW
11. ✅ Установку и настройку Fail2Ban
12. ✅ Импорт структуры базы данных

## 📝 Интерактивные настройки

Во время установки скрипт запросит:

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| Домен/IP | Адрес вашего сервера | localhost |
| Email | Для SSL-сертификата | admin@домен |
| Порт | Веб-порт | 80 |
| Директория | Путь установки | /var/www/reduce_links |

## 🛠️ Ручная настройка (если нужно)

### Изменение настроек базы данных

```bash
nano /var/www/reduce_links/includes/config.php
```

### Перезапуск Apache

```bash
sudo systemctl restart apache2
```

### Просмотр логов

```bash
# Логи Apache
sudo tail -f /var/log/apache2/reduce_links_error.log

# Логи MySQL
sudo tail -f /var/log/mysql/error.log
```

## ❌ Удаление

Для полного удаления выполните:

```bash
# Остановить и удалить Apache сайт
sudo a2dissite reduce_links
sudo rm /etc/apache2/sites-available/reduce_links.conf

# Удалить файлы
sudo rm -rf /var/www/reduce_links

# Удалить базу данных и пользователя
sudo mysql -u root -p -e "DROP DATABASE url_shortener;"
sudo mysql -u root -p -e "DROP USER 'urlshortener_xxxx'@'localhost';"

# Перезапустить Apache
sudo systemctl restart apache2
```

## 🆘 Решение проблем

### Ошибка прав доступа

```bash
sudo chown -R www-data:www-data /var/www/reduce_links
sudo chmod -R 755 /var/www/reduce_links
```

### Ошибка подключения к БД

```bash
# Проверить статус MySQL
sudo systemctl status mysql

# Перезапустить MySQL
sudo systemctl restart mysql
```

### 404 ошибка на ссылках

```bash
# Проверить включен ли mod_rewrite
sudo a2enmod rewrite
sudo systemctl restart apache2
```

При возникновении проблем:

3. Создайте issue на GitHub: https://github.com/pachugonis/reduce_links/issues
